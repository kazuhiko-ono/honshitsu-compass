## 写真の使用方針

Unsplashの写真を各セクションに適切に使い、魅力的なサイトにすること。

| セクション | 写真のイメージ | Unsplash検索キーワード |
|---|---|---|
| ヒーロー | 広大な海・空・地平線 | \`ocean horizon freedom\` |
| 私のこと | 静かな瞑想・内省 | \`reflection silence contemplation\` |
| 一緒に見つける | 対話・会話・並んで歩く | \`conversation walking together\` |
| 副操縦士エピソード | コックピット・空 | \`cockpit pilot sky\` |
| サービス | 穏やかな自然・光 | \`calm nature light\` |

- 写真はすべてUnsplashから取得（\`https://images.unsplash.com/photo-xxxxx?w=1200&q=80\`）
- 人物写真は使わない（特定のイメージを押しつけない）
- 写真はテキストの邪魔をしない配置にする
- ヒーローは背景画像として全幅表示
